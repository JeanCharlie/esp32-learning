Progressive ESP32 projects from basics to smart home, combining C++ firmware with React Native mobile apps.

## Structure

Each folder is a self-contained project with its own README, code, and wiring instructions.

| Project | Description |
|---|---|
| [01-blink-led](./01-blink-led) | Blink the built-in LED |

## Hardware
- Freenove ESP32-WROOM Board
- ELEGOO 37-in-1 Sensor Kit
- 5V Relay Module
- Raspberry Pi 4 (for Home Assistant)

## Stack
- Firmware: C++ with Arduino IDE
- Mobile app: React Native + Expo + TypeScript
