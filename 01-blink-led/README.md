Primer programa para ESP32. Hace parpadear el LED integrado cada 500ms.

## Hardware necesario
- ESP32 (cualquier variante con LED integrado en pin 2)

## Qué aprenderás
- Subir código al ESP32 con Arduino IDE
- Configurar pines GPIO como salida
- Función setup() y loop()

## Cómo usarlo
1. Abre main.ino en Arduino IDE
2. Selecciona ESP32 Dev Module en Tools → Board
3. Sube el código
4. El LED azul parpadea cada medio segundo
